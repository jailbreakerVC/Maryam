# Web search interface for OWASP Maryam IRIS module

prerequisites:
install maryam:
`pip install maryam`
run command
`maryam -e web api 127.0.0.1 1313`
then
clone this repository
`cd irissearch`
then `npm install`

and finally
`npm run dev`

app will be served at `http://localhost:5173/`
